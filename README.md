# McDonald’s Sales Analysis Project

## Overview

This project focuses on analyzing McDonald’s sales and order data to identify customer ordering patterns, revenue trends, category performance, and peak business hours. The project includes data preprocessing, exploratory data analysis (EDA), and dashboard-ready insights using Power Query, Excel .

## Objectives

* Analyze customer purchasing behavior
* Identify high-performing food categories
* Track revenue trends and sales performance
* Compare weekend vs weekday sales
* Detect peak and low sales periods
* Create an analysis-ready dataset for visualization

## Dataset Information

The project contains two datasets:

* **Menu Item Dataset** – Contains food item details, category, and pricing
* **Order Details Dataset** – Contains transactional sales records

## Data Preprocessing

The following preprocessing steps were performed:

* Header promotion
* Data type transformation
* Date format standardization
* Null value handling
* Removing blank rows and unnecessary columns
* Data merging using joins
* Revenue calculation
* Feature engineering for:

  * Month Name
  * Day Name
  * Revenue Segment
  * Weekend/Weekday Analysis

## Exploratory Data Analysis (EDA)

Key analyses performed:

* Revenue analysis
* Category-wise performance
* Peak sales hour analysis
* Weekend vs weekday comparison
* Day segment analysis
* High sales vs low sales trends

## Tools & Technologies Used

* Microsoft Excel
* Power Query

## Key Insights

* Evening and lunch hours generate the highest sales
* Burgers and beverages contribute significantly to revenue
* Weekend orders are higher compared to weekdays
* Revenue trends help identify high-performing menu items

## Conclusion

This project successfully transformed raw McDonald’s sales data into a clean and analysis-ready dataset, enabling meaningful business insights and interactive dashboard creation for better decision-making.
